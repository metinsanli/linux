Nasıl kurulur?

~$ sudo apt-get update
~$ apt-get install unzip
~$ wget download.java.net/glassfish/4.0/release/glassfish-4.0.zip
~$ sudo unzip glassfish-4.0.zip -d /opt
~$ cd /opt/glassfish4/bin
/opt/glassfish4/bin$ sudo ./asadmin start-domain

Bu işlemden sonra http://localhost:8080 adresine giderek kontrol edin.

 

Otomatik başlangıç için Servis haline getirme?

 

~$ nano -w glassfish

Açılan editör ekranına aşağıdaki kodları ekleyin.

#!/bin/sh

#

# glassfish init script for Linux

# Simplest possible case -- no password file, one default domain

# it would be simple to add such options

GLASSFISH_HOME=${GLASSFISH_HOME:-"/opt/glassfish4/glassfish"}

case "$1" in

start)

    $GLASSFISH_HOME/bin/asadmin start-domain >/dev/null

    echo "* Glassfish servisi baslatildi."

    ;;

stop)

    $GLASSFISH_HOME/bin/asadmin stop-domain >/dev/null

    echo "* Glassfish servisi durduruldu!"

    ;;

restart)

    $GLASSFISH_HOME/bin/asadmin restart-domain >/dev/null

    echo "* Glassfish servisi yeniden baslatildi."

    ;;

\*)

    echo "usage: $0 (start|stop|restart|help)"

esac

Eklemeden sonra sırasıyla; Ctrl+O ardından Enter tuşlarına basarak kaydedin. Ctrl+X tuşlarına basarak editörden çıkın.

Şimdi glassfish isimli (uzantısı olmayan!) bash script dosyasını kullanıyoruz.

~$ sudo cp glassfish /etc/init.d/
~$ cd /etc/init.d
/etc/init.d$ sudo chmod +x glassfish
/etc/init.d$ sudo update-rc.d glassfish defaults
/etc/init.d$ sudo reboot

 

Glassfish Server "admin" şifresi nasıl değiştirilir?

~$ cd /opt/glassfish4/bin
/opt/glassfish4/bin$ sudo ./asadmin change-admin-password

 

Farklı bir IP'den Glassfish Server'a erişim engelinin kaldırılması.

localhost haricinde başka bir bilgisayardan (dolayısıyla farklı bir IP'den) http://localhost:4848 yönetim konsoluna bağlanıp kullanıcı adı ve şifre ile giriş yapmayı denediğinizde izin verilmediğini göreceksiniz. Bu engelin kaldırılması için enable-secure-admin fonsiyonunu devreye almanız gerekiyor.

~$ cd /opt/glassfish4/bin
/opt/glassfish4/bin$ sudo ./asadmin enable-secure-admin

 

Servisi manuel başlatma/durdurma nasıl yapılır?

~$ sudo service glassfish start

~$ sudo service glassfish stop
~$ sudo service glassfish restart
